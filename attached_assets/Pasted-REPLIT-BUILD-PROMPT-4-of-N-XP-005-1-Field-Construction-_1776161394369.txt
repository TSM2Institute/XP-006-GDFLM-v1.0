STEP 1: Build galaxy surface density field chi(x,y)

  a) Load galaxy catalogue (RA, Dec)
  
  b) Convert RA, Dec to pixel coordinates on the 200x200 grid:
     - Grid centre pixel (100, 100) corresponds to the cluster centre
     - Use the RA/Dec centre from grid_params.json
     - Convert angular offset to pixel: offset_pixels = offset_arcsec / pixel_scale
     - RA increases to the left (west) in standard orientation — be careful 
       with sign convention. Use: x_pixel = centre_x - (RA - RA_centre) * 3600 * cos(Dec_centre_rad) / pixel_scale
       y_pixel = centre_y + (Dec - Dec_centre) * 3600 / pixel_scale
  
  c) Create 2D histogram of galaxy positions on the grid (200x200)
  
  d) Smooth with Gaussian kernel:
     - sigma = SIGMA_GAL_ARCSEC / pixel_scale_arcsec (from manifest)
     - This should be approximately 4.53 / 2.0 = 2.27 pixels
     - Use scipy.ndimage.gaussian_filter
  
  e) Normalise to [0, 1]:
     chi = smoothed_density / max(smoothed_density)
     Handle edge case: if max == 0, chi = zeros

  f) Save: data/processed/chi_field.npy
