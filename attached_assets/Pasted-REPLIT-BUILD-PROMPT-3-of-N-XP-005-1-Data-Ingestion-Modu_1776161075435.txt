--- PART B: Parametric Density and Temperature Fields ---

We use published β-model parameters instead of digitising figures.
This is the manifest-approved fallback (Manifest §4.1: "β-model fallback").

Create a 2D grid centred on the Bullet Cluster. The grid represents the 
projected sky plane.

GRID SETUP:
- Grid size: 200 x 200 pixels
- Pixel scale: 2 arcsec/pixel (covers ~6.7' x 6.7', sufficient for cluster)
- Centre: (100, 100) = cluster centre
- Bullet subcluster offset: ~65 arcsec east of main centre → ~33 pixels at 
  2 arcsec/pixel. Place bullet centre at pixel (133, 100).

ELECTRON DENSITY — Double β-model (two components):
Published parameters from Markevitch et al. 2002 (ApJ 567, L27) and 
Markevitch 2006 (ESA SP-604):

  Component 1 — Main cluster:
    n_0_main = 3.0e-3    # cm^-3, central electron density
    r_c_main = 250.0     # kpc, core radius
    beta_main = 0.70     # β parameter

  Component 2 — Bullet subcluster:
    n_0_bullet = 4.0e-2  # cm^-3, central electron density  
    r_c_bullet = 50.0    # kpc, core radius
    beta_bullet = 0.60   # β parameter

  β-model formula (projected, cylindrical):
    n_e(r) = n_0 * (1 + (r/r_c)^2)^(-3*beta/2)

  Total field: n_e_total(x,y) = n_e_main(r_main) + n_e_bullet(r_bullet)
  where r_main = distance from (100,100) and r_bullet = distance from (133,100),
  both converted from pixels to kpc using KPC_PER_ARCSEC and the pixel scale.

  Then convert: rho(x,y) = n_e_total * M_PROTON (from manifest)

TEMPERATURE — Two-component model:
Published values from Markevitch 2006:

  T_main = 14.1     # keV, main cluster average
  T_bullet = 6.5    # keV, bullet subcluster  
  T_background = 10.0  # keV, ambient/average

  Implementation:
    T(x,y) = T_background
    Add main cluster: Gaussian centred at (100,100), amplitude (T_main - T_background),
    width = 60 pixels (~120 arcsec ~ 530 kpc)
    Add bullet: Gaussian centred at (133,100), amplitude (T_bullet - T_background),
    width = 20 pixels (~40 arcsec ~ 177 kpc)
    
    This creates a temperature drop at the bullet location, with a gradient 
    between the hot main cluster and cool bullet — matching the observed 
    Bullet Cluster temperature structure.

  Compute: grad_T = numpy.gradient(T), then magnitude |grad_T|

SURFACE BRIGHTNESS (for baseline):
  SB(x,y) proportional to n_e^2 (standard X-ray bremsstrahlung):
    SB = n_e_total^2

  This is the observationally motivated baseline.

IMPORTANT — Record all parametric model parameters in a dictionary at the 
top of the script with full citations:

  LITERATURE_PARAMS = {
      'n0_main': {'value': 3.0e-3, 'units': 'cm^-3', 
                  'source': 'Markevitch et al. 2002, ApJ 567, L27'},
      'rc_main': {'value': 250.0, 'units': 'kpc',
                  'source': 'Markevitch et al. 2002, ApJ 567, L27'},
      # ... etc for all parameters
  }

Print these when the script runs so they're in the log.
