=== TASK ===
Create src/mock_validation.py — a self-contained script that builds synthetic 
2D fields, runs them through every manifest equation, and verifies the outputs 
behave as expected.

The mock uses a simple 100x100 pixel grid with fake but physically motivated fields.
This is NOT a test on real data — it's a sanity check that the math works.

STEP 1: Create synthetic input fields (100x100 grid)

  a) Electron density field n_e(x,y):
     - Place a Gaussian "cluster" at grid centre (50,50)
     - Peak n_e = 0.1 cm^-3 (typical ICM core)
     - Width = 15 pixels
     - Background n_e = 0.001 cm^-3
     - Convert to mass density: rho = n_e * M_PROTON

  b) Temperature field T(x,y):
     - Hot core: 14 keV at centre (Bullet main cluster)
     - Cool "bullet": 6 keV Gaussian at offset position (65,50), width=8 pixels
     - Background: 10 keV
     - This creates a temperature gradient between hot and cool regions
     - Compute grad_T using numpy.gradient, then magnitude |grad_T|

  c) Galaxy surface density chi(x,y):
     - Two Gaussian peaks (representing the two mass concentrations)
     - Peak 1 at (45,50), width=12 pixels (main cluster galaxies)
     - Peak 2 at (65,50), width=8 pixels (bullet galaxies)  
     - Normalise to [0,1]: chi = Sigma_gal / max(Sigma_gal)

STEP 2: Run the forward model using manifest functions

  From manifest.py, import and call in order:
  1. P = phase_function(rho)
  2. psi_emit = boundary_emission(P, chi)
  3. psi_prop = propagation_field(P, grad_T_magnitude, chi)
  4. psi_obs = observed_field(psi_emit, psi_prop)
  5. kappa_model = model_convergence(psi_obs)

  Also create a fake surface brightness (proportional to n_e^2, 
  standard X-ray emission) and compute:
  6. kappa_baseline = baseline_convergence(fake_SB)

STEP 3: Run verification checks (print PASS/FAIL for each)

  CHECK 1 — Phase function peaks at rho_threshold:
    P should be highest where rho is closest to rho_threshold.
    Since our mock has rho >> rho_threshold everywhere (ICM densities 
    ~1e-28 to ~1e-25 g/cm3), P should peak in the outskirts where 
    rho approaches rho_threshold, not at the centre.
    VERIFY: Location of max(P) is NOT at grid centre (50,50).

  CHECK 2 — Psi_emit responds to both P and chi:
    VERIFY: max(psi_emit) > 0 and psi_emit is not uniform.

  CHECK 3 — Psi_prop responds to temperature gradient:
    VERIFY: psi_prop is highest where |grad_T| AND P AND chi overlap.
    psi_prop should be near-zero where grad_T is zero.

  CHECK 4 — kappa_model is normalised to [0,1]:
    VERIFY: max(kappa_model) == 1.0 and min(kappa_model) >= 0.0

  CHECK 5 — kappa_baseline is normalised to [0,1]:
    VERIFY: max(kappa_baseline) == 1.0 and min(kappa_baseline) >= 0.0

  CHECK 6 — kappa_model != kappa_baseline:
    VERIFY: The model and baseline produce DIFFERENT spatial distributions.
    This confirms the propagation field (grad_T, chi) adds information 
    beyond raw surface brightness.

  CHECK 7 — Zero-input safety:
    Run model_convergence(np.zeros((10,10))) — should return all zeros, no crash.
    Run baseline_convergence(np.zeros((10,10))) — same.

  CHECK 8 — P(x) value range:
    Print min(P), max(P), mean(P).
    All values should be in [0, 1].
    If max(P) is very small (< 0.01), flag this — it means rho is 
    nowhere near rho_threshold on this grid, which is physically 
    informative (and will be relevant for the real data too).
